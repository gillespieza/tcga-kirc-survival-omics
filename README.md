# TCGA KIRC Survival Omics Mini-Project

This repository contains code and report materials for a mini-project integrating clinical and ~omics data to predict overall survival in clear cell renal cell carcinoma (ccRCC).

The project uses TCGA Kidney Renal Clear Cell Carcinoma (KIRC) data from cBioPortal. The planned analysis integrates clinical variables with RPPA proteomic features and genomic alterations, focusing on inflammatory invasion and protein tyrosine kinase (PTK) signalling biology.

## Planned outputs

- Reproducible R code for data retrieval and analysis
- Survival model comparison
- Figures and tables for a brief report
- Biological interpretation of selected clinical and ~omics features
